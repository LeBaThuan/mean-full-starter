A Modern Guide for the MEAN Stack


Welcome to MEANkit.io, the official website of the MEAN Stack Full Starter Template. MEAN is a full-stack development toolkit using the Javascript-based technologies MongoDB, Express, Angular, and Node.js. This template is one of the best resources for your next web development project. Many of the necessary features needed for today's modern web apps are directly built-in for immediate use, including authentication with JSON Web Tokens, cloud database service with MongoDB Atlas, and cloud storage with Amazon S3.


The Angular client was coded using best-practices with modular architecture, dependency injection, and lazy loading and was designed using Bulma, an open source CSS-only framework based on Flexbox. The Stripe API has also been integrated into the app and serves as a starting point for the further development of a user payment system. In addition to all of these features, the website takes advantage of Angular Universal for SEO and social media compatibility.


Detailed explanations are provided as a guide to help you integrate these features into your own projects. Feel free to explore this website to get a feel for what went in to making it and if you're extra curious, the source code can be downloaded from the GitHub link on the top left of the screen. So what are you waiting for? Let's get started building the next great app today!

Setup Guides

Authentication with JWT

JSON Web Token (JWT) is an open standard (RFC 7519) that defines a compact and self-contained way for securely transmitting information between two parties as a JSON object. Authentication is the most common use scenario. Each request after login will include the JWT for further web resource access.

MongoDB Atlas

MongoDB Atlas is a cloud database service that stores data in flexible, JSON-like documents, and makes it easy to deploy, operate, and scale MongoDB in the cloud by automating time-consuming administration tasks such as database setup, security implementation, scaling, patching, and more.

Amazon S3 Cloud Storage

Amazon S3 Storage is a platform to easily and securely collect and store data on a massive scale. Its features allow users to take a data-driven approach to storage optimization, compliance, and management efficiency. These features work together to help improve workload performance and storage costs.

Deploying the App using Cloud Services

Once the app has been completed, it needs to be deployed to the internet to be enjoyed by all! Fortunately, there are many platform-as-a-service (PaaS) providers who make deployment easier than ever before! Guides to deploy this MEAN stack web application are provided below for three popular hosting providers: Heroku, Amazon Web Services (AWS), and Zeit Now. Each service has their own strengths and weaknesses so the choice is up to you!
